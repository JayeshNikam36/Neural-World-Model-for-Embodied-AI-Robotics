# Dataset Documentation

## Selected Dataset: RoboNet

### Source
- Name: RoboNet
- URL: https://www.robonet.wiki/
- Paper: https://arxiv.org/abs/1910.11215

### Dataset Description
- **Size**: ~15 million frames
- **Robots**: 7 different robot platforms
- **Tasks**: Pick and place, pushing, pouring, etc.
- **Format**: HDF5 files
- **Split**: We'll use pre-split train/val/test
